Python Collections (Arrays)
There are (four collection data types) in the Python programming language:

#List and tuple (allow) duplicate values:

List -> is a collection which is ordered and changeable, indexable. Allows duplicate members.
Tuple-> is a collection which is ordered and unchangeable. Allows duplicate members.

#Set and Dictionary doesn't allowed any duplicate members:

Set ->is a collection which is unordered and unindexed, unchangeable. No duplicate members.
Dictionary-> is a collection which is also unordered but changeable. No duplicate members.
